# Compiling OFS on Fedora 9

## Required Packages

* `gcc-c++`
* `autoconf`
* `automake`
* `fuse-devel`
* `pkgconfig`
* `libtool`
* `libattr`
* `attr`
* `libconfuse-devel`
* `dbus-devel`
* `libattr-devel`

For creating the doxygen documentation you need

* `doxygen`
* `graphviz`
